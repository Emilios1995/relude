Relude_Eff

This module does not have a toplevel documentation block.

type t('a) = unit => 'a

Eff is a synchronous effect monad.

This is inspired by bs-effects Effect, purescript Eff and Effect, and John De Goes' basic synchronous IO monad described here: http://degoes.net/articles/only-one-io

let run: t('a) => 'a

Unsafely runs the effect or chain of effects. For Eff, this is the same as just calling the effect as a function.

let pure: 'a => t('a)

let fromThunk: unit => 'a => t('a)

let attempt: t('a) => t(Belt.Result.t('a, exn))

Attempts to run an Eff.t('a) and catches any exn thrown by the effect and lifts it into Eff.t(Belt.Result.t('a, exn))

let attemptJS: t('a) => t(Belt.Result.t('a, Js.Exn.t))
