Relude_Validation

This module does not have a toplevel documentation block.

type t('a, 'e) = | VOk('a) | VError('e)

Similar to Belt.Result, but has an Applicative instance that collects the errors using a semigroup, rather than fail-fast semantics.

let flatMapV: (t('a, 'e), 'a => t('b, 'e)) => t('b, 'e)

This function performs a flatMap-like operation, but if the f fails, all previous errors are discarded. Validation is not a traditional Monad, because the point of it is to preserve the errors via a Semigroup.
