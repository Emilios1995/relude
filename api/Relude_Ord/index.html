let clampBy: a. compare('a) => min:'a => max:'a => 'a => 'a;

Ensure a provided value falls between a max and min (inclusive). Note that if the provided min is greater than the provided max, the max is always returned. This is considered an incorrect use of `clamp`.

``` let clamp = clampBy(Int.compare); clamp(~min=0, ~max=5, 3) == 3; clamp(~min=0, ~max=5, 0) == 0; clamp(~min=0, ~max=3, 4) == 3; clamp(~min=1, ~max=0, 2) == 0; // don't do this ```
